Documentation

I have a great interest in retro consoles. Mostly the Sega Genesis. I also liked how stuff worked; so I made this documentation for my own use and maybe for other people's use. This documentation is about the Motorola 68000, AKA: the CPU used in the Sega Genesis. This page of the documentation most likely won't tell you anything about the microcontroller, but instead would link you to other pages about the microcontroller.
